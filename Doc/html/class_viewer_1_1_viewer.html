Bridge Gap Detection With OpenCV-Python and PyRealsense2
Project to control a fully autonomous drone for the deployment of a ground robot on a beidge support pillar, between the bridge bearings. This script processes the colour and depth data from the Intel Realsense D435i camera, and outputs the movement control commands, alongside distances, for the control of the drone.

def Viewer.Viewer.verify_deployment_area (   self,
  shrunk_points 
)

Function to verify whether the edges of the internal contour cross any point of the surrounding support structure.

If the edges do cross the structure, the drone cannot deploy as it will collide with pillar. If they do not, then there is a verified empty space for the drone to deploy the ground robot within.

Parameters

shrunk_points: The shrunken inner contour of the gap.
